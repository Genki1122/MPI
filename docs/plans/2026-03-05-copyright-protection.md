# 著作権保護・サイトポリシー 実装プラン

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** MPIサイトのコンテンツ（テキスト・画像・ロゴ）を著作権侵害から保護する。法的な「サイトポリシー」ページの新設と、技術的なコピー防止策を実装する。

**Architecture:** (1) サイトポリシーページ新設、(2) フッターに著作権表示強化＋ポリシーリンク追加、(3) CSS/JSによるコピー防止、(4) 印刷制限。

**Tech Stack:** HTML / CSS / JS（既存構造拡張）

---

## クライアント要望の分析

クライアントの要望を整理すると3点：

| 要望 | 対応 |
|------|------|
| 「無許可の転載を防ぐ言葉」 | → サイトポリシーページ新設（法的文言） |
| 「コピーしにくい仕様」 | → CSS/JSでテキスト選択・右クリック・ドラッグ禁止 |
| 「別建ての事務事項」 | → フッターからリンクする独立ページとして構成 |

**注意点:**
- プライバシーポリシー（相手の個人情報保護）ではなく、**自社コンテンツの知財保護**が目的
- IPマネジメントを謳う会社として模範的な知財保護を示す
- 堅苦しくなく、しかし「これがあればパクリできない」実効性

## タスク

### Task 1: サイトポリシーページ HTML 新設

**Files:**
- Create: `site/policy/index.html`

日本の大手企業（日本総研・電通・博報堂・講談社）のサイトポリシーを参考に、MPI向けの文案を配置する。ページ名は「サイトポリシー / Site Policy」（最も一般的で格式がある名称）。

**構成:**
1. 著作権について — コンテンツ所有権の宣言
2. コンテンツのご利用について — 無断転載・複製の禁止
3. 引用について — 著作権法上の例外ルール
4. 商標・ロゴについて — 商標使用制限
5. 第三者のコンテンツについて — 他社との協業コンテンツの帰属
6. 法的措置について — 侵害への対応警告
7. 免責事項 — サイト情報の正確性に関する免責

### Task 2: フッター改修（全ページ共通）

**Files:**
- Modify: `site/index.html`
- Modify: `site/core/index.html`

**変更内容:**
- `© MPI Co., Ltd.` → `© 2004 MPI Co., Ltd.`（設立年追加）
- サイトポリシーへのリンク追加
- フッターに「無断転載禁止」の一行メッセージ追加

### Task 3: CSS — コピー保護＋印刷制限

**Files:**
- Modify: `site/css/style.css`

**変更内容:**
- `user-select: none`（テキスト選択禁止、フォーム要素は除外）
- 画像のドラッグ防止（`-webkit-user-drag: none`, `pointer-events: none`）
- 印刷時の制限（コンテンツ非表示＋著作権メッセージのみ表示）
- フッターのポリシーリンクスタイル

### Task 4: JS — コピー保護スクリプト

**Files:**
- Modify: `site/js/main.js`

**変更内容:**
- 右クリック（コンテキストメニュー）禁止
- コピー操作のインターセプト
- 画像ドラッグ防止
- Ctrl+C / Ctrl+U / Ctrl+S のキーボードショートカット制限

**UXへの配慮:**
- フォーム入力欄（Contact）のテキスト選択は許可
- 印刷（Ctrl+P）は完全ブロックせず、印刷用CSSで著作権表示を挿入

### Task 5: INDEX にサイトポリシーリンク追加

**Files:**
- Modify: `site/index.html`

INDEXグリッドには追加しない（事務的ページのため）。フッター内にのみリンクを配置する。

## © / ® / TM の使い分け

| マーク | 意味 | MPIでの使用 |
|--------|------|-------------|
| © | 著作権表示 | フッターに使用（法的には不要だが抑止力） |
| ® | 登録商標 | 特許庁に商標登録済みの場合のみ使用可（未登録で使うと刑事罰） |
| TM | 商標主張 | 未登録の場合はこちらを使用（法的罰則なし） |

**→ クライアントに確認:** MPIの商標は特許庁に登録済みか？

## 技術的保護の限界について

CSS/JSによる保護は**心理的抑止力**であり、完全な防御ではない：
- 開発者ツールで無効化可能
- スクリーンショットは防げない
- ブラウザ拡張機能で回避可能

しかし、**IPマネジメント企業として「保護の姿勢を示す」こと自体が重要**。法的文言（サイトポリシー）が本体であり、技術的保護はその補強。
